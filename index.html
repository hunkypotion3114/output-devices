Output Devices

Where technology & craftsmanship meet.

Output devices are pieces of computer hardware that allow a computer system to communicate information to a user or another system. Information can be in any form, and includes sound and images. Output devices can usually only be used to send data from the computer.

Speakers

A computer speaker is an output hardware device that generates sound.Signal used to produce the sound is created by the computer's sound card.

Projector

A projector is an output device that takes images generated by a computer or Blu-ray player and reproduce them by projection onto a screen, wall, or another surface.

Learn More

Monitor

Alternatively referred to as VDT(video display terminal) and VDU(video display unit), a monitor is an output device that displays video images and text.

Printer

A printer is an external hardware output device that takes the electronic data stored on a computer or other device and generates a hard copy.

Plotter

A plotter like a printer that is used for printing vector graphics. Plotters use a pen, pencil, marker, or another writing tool to draw multiple, continuous lines onto paper.
